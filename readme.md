# 🎓 Faculty Appointments 2026 – YCCE (Web Replica)

## 📌 Project Overview

This project is a **frontend replica** of the Faculty Appointments page of YCCE (Yeshwantrao Chavan College of Engineering). It is built using **HTML and CSS only** to demonstrate webpage structure, layout design, and styling techniques.

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

## 📂 Project Structure

```
project-folder/
│── index.html
│── style.css
│── images/
     └── ycce.png
```

---

## 🎯 Features

* Top **red notice bar** with latest announcements
* **"NEW" badges** styled using CSS
* Right-side **Notice Board panel**
* Hero section with **blurred background image**
* Overlay text with heading and navigation link
* Styled **buttons for application actions**
* Clean layout and alignment similar to original webpage

---

## 🖼️ Background Image Usage

* A screenshot (`ycce.png`) is used as background
* CSS **blur effect** is applied to remove visible text
* Dark overlay ensures readability of new content

---

## 🚀 How to Run the Project

1. Open **Visual Studio Code**
2. Create a folder and add:

   * `index.html`
   * `style.css`
   * `images/ycce.png`
3. Open `index.html` in browser

---

## ⚙️ Key CSS Concepts Used

* Flexbox (for layout alignment)
* Background image styling
* Blur filter (`filter: blur()`)
* Overlay using `rgba()`
* Button styling and spacing
* Responsive alignment basics
